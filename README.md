# GROCEY LIST

## AUTHOR
Neville Oronni

## DESCRIPTION
Create a form that allows a user to input items they need at the store.
When the user submits the form, hide the form and display the list of items in alphabetical order, all uppercased.

## TECHNOLOGIES USED  
* HTML
* CSS
* JQUERY
* BOOTSTRAP
* JAVASCRIPT
